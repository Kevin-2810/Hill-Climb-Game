# Hill-Climb-Game
A fun and interactive Hill Climb Racing-inspired browser game built using HTML, CSS, and JavaScript.
Choose different vehicles, drive through challenging hills, collect coins, manage fuel, and survive as long as possible!

🚀 Features

🎮 Smooth hill climbing gameplay

🚙 Multiple vehicle selection

Jeep
Truck
Bike
Buggy
Tractor
Bus

⛽ Fuel management system

🪙 Coin collection

📈 Speed & distance tracking

⚠ Low fuel warning

🔄 Flip detection (Game Over)

📱 Mobile touch controls

🎯 Difficulty modes
Easy
Medium
Hard

🏆 Best score tracking
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Canvas API

browser
🎮 Controls
Action	Key
Brake	← Left Arrow
Gas	→ Right Arrow
